# Simple TCP Echo Server

This is a Simple TCP Echo Server, i.e What ever is sent to server it responds with exact same thing.

### How to Run?
- Use `telnet` (on windows) or `netcat (nc)` (on Linux)
```
    `telnet localhost 7280`
```

OR

```
    `nc localhost 7280`
```
